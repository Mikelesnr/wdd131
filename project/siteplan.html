Site Plan - ZoomCar Rentals

Site Name

ZoomCar Rentals

This name was selected to convey the speed and convenience of our car rental service.

Site Purpose

The site provides a platform for users to view available cars, check rental rates, book cars, and read customer reviews.

Scenarios

  • What cars are available for rent?
  • How can I book a car online?

Color Schema

Primary Color: #9e6240 (used for headings and navigation)

Secondary Color: #E1306C (used for accents and buttons)

Background Color: #FCB045 (used for the page background)

Text Color: #423c3b (used for body text)

Typography

Font 1: Roboto (used for body text)

Font 2: Merriweather (used for headings)
